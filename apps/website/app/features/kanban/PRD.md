# PRD: 채용 칸반 보드

## Iteration 1

### 목표

채용 전형별 칸반 보드 페이지를 구현한다. 하나의 채용 공고에 여러 전형(서류심사, 1차 면접, 2차 면접, 최종합격)이 존재하고, 지원자 카드를 드래그하여 전형 간 이동하거나 컨텍스트 메뉴로 액션을 수행할 수 있다. 이 이터레이션에서는 더미 데이터를 사용한다.

### 기능 명세

1. **칸반 보드 표시**
   - 채용 공고 정보(제목, 부서, 상태)를 헤더에 표시
   - 전형별 컬럼을 가로로 나열 (서류심사 → 1차 면접 → 2차 면접 → 최종합격)
   - 각 컬럼 헤더에 전형 이름과 해당 전형의 지원자 수를 표시

2. **지원자 카드**
   - 아바타, 이름, 태그(경력/신입 등) 표시
   - 우클릭 시 컨텍스트 메뉴 표시
   - 카드 내 더보기 버튼 클릭 시 드롭다운 메뉴 표시
   - 메뉴 항목: 메시지 보내기, 메모 추가, 이력서 보기, 탈락 처리

3. **드래그 앤 드롭**
   - 지원자 카드를 드래그하여 같은 컬럼 내 순서 변경
   - 지원자 카드를 드래그하여 다른 컬럼으로 이동
   - 드롭 위치에 인디케이터 표시
   - 빈 컬럼에도 드롭 가능

### UI/UX 명세

- **레이아웃**: 전체 화면 너비 사용, 가로 스크롤 가능
- **보드 헤더**: 채용 제목(h1), 부서명, 상태 Badge
- **컬럼**: 고정 너비(320px), 세로 스크롤 가능한 카드 리스트
- **카드**: Card 컴포넌트 사용, hover 시 약간의 그림자 변화
- **드래그 피드백**: 드래그 중 카드 반투명(opacity-50), 드롭 타겟 컬럼 배경색 변경, 드롭 인디케이터(파란색 라인)
- **컨텍스트 메뉴**: 우클릭으로 접근, DropdownMenu와 동일한 메뉴 항목
