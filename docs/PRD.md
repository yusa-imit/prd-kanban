## Iteration 4: 가상화 & Optimistic Update 인프라

### 목표

12개 전형 × 1,000명 규모의 대규모 데이터를 가상화로 처리하고, API 도입을 대비한 optimistic update 패턴을 구축한다.

### 기능 명세

#### 4-1. 대규모 더미 데이터 생성

- 12개 전형: 서류심사, 코딩테스트, 1차 면접, 과제전형, 2차 면접, 임원면접, 레퍼런스체크, 처우협상, 건강검진, 최종합격, 입사대기, 입사완료
- 각 전형 1,000명 (총 12,000명)
- 한국 이름 풀(성 20 × 이름 20) + 스킬 태그를 deterministic하게 조합

#### 4-2. Optimistic Update 인프라

- `useBoardMutation` 훅: snapshot → optimistic apply → (optional) API → rollback on failure
- 순수 함수로 상태 변환 로직 분리
- `apiFn` 없으면 동기 동작 (기존 동작과 동일)
- API 통합 시 `apiFn`만 추가하면 optimistic update 완성

#### 4-3. 컬럼 가상화

- `@tanstack/react-virtual`의 `useVirtualizer`로 컬럼 내 카드 가상화
- Radix ScrollArea → plain div + `overflow-y: auto`
- `autoScrollForElements`로 드래그 중 세로 자동 스크롤
- absolute positioning으로 가상 아이템 렌더링

#### 4-4. 보드 레이아웃 최적화

- `monitorForElements` deps에서 `boardState` 제거 → `boardStateRef` 사용
- 보드 가로 스크롤 영역에 `autoScrollForElements` 등록

### UI/UX 명세

- 12개 컬럼이 가로 스크롤로 표시됨
- 각 컬럼 내 1,000개 카드가 가상화되어 부드러운 스크롤
- 드래그 중 세로(컬럼 내) + 가로(보드) 자동 스크롤
- 기존 체크박스 선택, 벌크 드래그, 로그 패널 모두 유지

### 마일스톤 기준

- [x] 12개 컬럼 × 1,000명 렌더링 확인
- [x] 스크롤 시 부드러운 가상화 동작
- [x] 가상화 리스트에서 단건 드래그앤드롭 동작
- [x] 가상화 리스트에서 벌크 드래그앤드롭 동작

- [x] 체크박스 선택/벌크 툴바 기존 기능 유지

## Iteration 5: 전형 상세보기 — 액션별 칸반보드

### 목표

메인 칸반보드의 각 전형(서류심사, 코딩테스트 등) 내부에 세부 액션 단계를 도입하고, 전형 헤더 클릭 시 액션별 칸반보드로 진입할 수 있도록 구현한다.

### 기능 명세

#### 5-1. 전형별 액션 데이터

- 12개 전형에 각각 3~5개의 세부 액션 단계를 정의
  - 서류심사: 접수완료 → 서류검토중 → 서류합격 → 결과통보
  - 코딩테스트: 시험발송 → 시험진행중 → 채점중 → 채점완료
  - 면접류: 일정조율 → 면접대기 → 면접완료 → 평가중
  - 기타 전형도 동일한 패턴으로 3~4개 액션 정의
- 각 지원자에게 `currentActionId`를 부여하여 현재 액션 단계 표시

#### 5-2. 메인 보드 카드 액션 표시

- 기존 카드 디자인(이름 + 태그 + 체크박스 + 메뉴) 유지
- 태그 아래에 현재 액션 정보를 한 줄 추가 (`→ 서류검토중` 형태, text-[11px], muted 컬러)

#### 5-3. 컬럼 헤더 링크

- 메인 보드의 전형 헤더(전형명)를 클릭하면 `/recruitment/:stageId`로 이동
- 호버 시 밑줄 표시

#### 5-4. 전형 상세 칸반보드

- 상단에 뒤로가기 버튼 + 전형명 + 지원자 수 표시
- 액션별 컬럼으로 구성된 칸반보드 렌더링
- 기존 `KanbanColumn`, `KanbanCard`, `BulkActionToolbar`, `DevPanel` 재사용
- 드래그앤드롭 (단건 + 벌크) 지원
- 가상화 스크롤 적용

### UI/UX 명세

- 메인 보드: 카드에 현재 액션이 작은 텍스트로 표시
- 메인 보드: 컬럼 헤더가 클릭 가능한 링크
- 상세 보드: 뒤로가기(`/recruitment`) + 전형명 + 지원자 수 헤더
- 상세 보드: 액션별 3~5개 컬럼이 가로 스크롤로 표시
- 상세 보드: 기존 칸반보드와 동일한 DnD 및 벌크 선택 기능

### 마일스톤 기준

- [x] 메인 보드 카드에 현재 액션 뱃지 표시
- [x] 메인 보드 컬럼 헤더 클릭 → `/recruitment/:stageId` 이동
- [x] 상세 보드: 액션별 컬럼 + 지원자 카드 렌더링
- [x] 상세 보드: 드래그앤드롭 (단건 + 벌크) 동작
- [x] 상세 보드: 가상화 스크롤
- [x] 상세 보드: 뒤로가기 → 메인 보드 복귀
